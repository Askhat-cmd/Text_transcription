## 6. Generic target behavior

Add a generic rule:

```text
If current turn is a direct conceptual question OR a follow-up to a just-discussed concept,
and selected relevant knowledge exists,
and no hard blocker applies,
then build a minimal writer-visible hidden knowledge package.
```

Recommended names, exact names may adapt:

```text
grounding_reason = direct_concept_followup
or grounding_reason = knowledge_continuation
or grounding_reason = selected_relevant_knowledge_no_hard_block
```

Important:

```text
no_clear_retrieval_need is valid for greeting/contact/thanks/close/ordinary support.
no_clear_retrieval_need is not valid when current turn asks "как влияет", "что говорится", "почему", "объясни" about a selected known concept.
```

The package must be minimal:

```text
- at least one best selected card/chunk to Writer;
- max 2-3 items unless user explicitly asks deeply;
- Writer can ignore = true;
- applied_as_authority = false;
- quote_policy and allowed_use enforced;
- public answer clean from internal language.
```
