## 4. Как изменена relation logic
В `ThreadManagerAgent._resolve_relation_with_debug(...)`:
- Добавлен `_LOW_RESOURCE_CONTINUE_MARKERS`.
- Добавлено правило `low_resource_continuation_marker` до проверки `continuity_below_threshold`.
- Branch marker сохранён сильнее low-resource guard (ветка `branch` проверяется раньше).
- Для low-resource continuation continuity поднимается `max(continuity_raw, 0.25)`.
