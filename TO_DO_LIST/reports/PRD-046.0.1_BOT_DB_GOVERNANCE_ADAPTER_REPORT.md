# PRD-046.0.1 Bot_data_base Governance Adapter Report

## What changed
- Added adapter entrypoint: `Bot_data_base/governance/governance_adapter.py`.
- Added chunking quality trace builder: `Bot_data_base/governance/chunking_quality.py`.
- Added pipeline integration after normalization and before export/indexing.

## Governance schema
Every `UniversalBlock` now stores:
- `governance.schema_version`
- `governance.chunk_type`
- `governance.allowed_use[]`
- `governance.safety_flags[]`
- `governance.lens_family[]`
- `governance.tags[]`
- `governance.practice_metadata{}`
- `governance.governance_notes[]`
- `governance.source_trace{...}`

Also stored per block:
- `chunking_quality.schema_version`
- `char_count`, `approx_token_count`
- `possible_practice_split`, `possible_context_fragment`
- `too_short`, `too_long`, `quality_notes[]`

## Profiles
- `general_book`
- `practice_manual`
- `architecture_notes`
- `transcript`

Invalid profile handling:
- API fallback to safe default (`general_book` for books, `transcript` for youtube).
- Job result includes warning text for invalid profile.

## Example synthetic blocks
Source: `synthetic_author__synthetic_governed_book`
- chunk_type counts: `theory=3`
- allowed_use counts: `diagnostic_lens=3`, `writer_context=3`, `practice_suggestion=3`
- safety_flags counts: `requires_grounding=3`, `not_for_direct_quote=3`

## JSON export
- Nested `metadata.governance` is present.
- Nested `metadata.chunking_quality` is present.

## Chroma metadata
Flat governance fields are present:
- `governance_schema_version`
- `governance_chunk_type`
- `governance_allowed_use` (CSV)
- `governance_safety_flags` (CSV)
- `governance_lens_family` (CSV)
- `governance_low_resource_safe` (string bool)
- `governance_not_for_direct_quote` (string bool)
- `governance_source_style_not_user_facing` (string bool)

## Query API
`ChunkResult` now includes optional `governance` summary built from flat Chroma metadata.

## Notes
- Runtime filtering is not enabled in this PRD.
- Writer/DiagnosticCenter behavior is unchanged.
