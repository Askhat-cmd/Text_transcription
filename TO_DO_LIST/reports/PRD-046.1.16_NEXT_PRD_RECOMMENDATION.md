# PRD-046.1.16 Next PRD Recommendation

## Recommended next PRD
- `PRD-046.1.17 - Diagnostic Center Response Quality Eval Pack v1`

## Why this is the safest next step
- PRD-046.1.16 already закрыл runtime governance acceptance и boundaries.
- Следующий шаг должен проверять качество пользовательских ответов, а не расширять runtime authority.

## Scope for PRD-046.1.17
- Curated live-like response quality scenarios.
- Метрики: точность удержания линии, бережность, неповторяемость, уместность глубины, качество микро-сдвига, отсутствие книжности.
- Отдельные проверки соблюдения KB/internal lens boundaries и no-raw-quote policy.

## Explicit non-goals for PRD-046.1.17
- Не включать broad runtime authority.
- Не расширять apply path для normal users.
- Не менять deterministic KB authority fields.
