## Dataset Changes
- Added `12` quality cases across groups `A/B/C/D`.
- Added explicit expected checks for short-support, imperfect-self mechanism framing, drivers soft mirror, and neurostalking internal-lens framing.

## Evaluator Changes
- Added answer-level checks:
  - `selected_lens_match`
  - `no_unsolicited_practice`
  - `no_external_surveillance`
  - `no_quote_source`
  - `no_diagnosis`
  - `no_hard_label`
  - `short_answer_when_requested`
  - `no_analysis_when_requested`
  - `mechanism_not_defect_frame`
  - `required_semantics_present`
  - `answer_not_too_technical`
  - `answer_not_too_generic`
  - `question_count_lte_1`
  - `prompt_compactness_within_budget`
- Failures return explicit `failure_reasons` per case.

## Kernel Selector Calibration
- Added/expanded aliases for:
  - imperfect-self (`прошивка`, `потерплю неудачу`)
  - drivers (`не буду сильным`)
  - short-support (`не нужны практики`, `побудь со мной коротко`)
- Added `autopilot` lens activation for inner-loop phrasing.
- Capped selected lenses to `<=3`.

## Writer Prompt Micro-Tuning
- Added guidance in writer system/user prompts:
  - speak from lens, not about lens
  - do not expose lens names unless useful
  - avoid bureaucratic phrasing for personal tone
  - warm direct style for vulnerable personal requests
- Added low-resource compliance clamp in Writer post-processing to keep short no-practice contact compact.
