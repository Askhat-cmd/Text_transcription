# PRD-047.5 Next PRD Recommendation

## Recommended Next PRD
`PRD-047.6 - Planner Runtime Drift Guard / Stability Envelope`

## Why
- PRD-047.5 reached `dry/direct/live = 26/26`, but live generation remains probabilistic and can drift with model/runtime noise.
- Current answer-fit gates are strong at acceptance time; next step should harden long-run stability and detect drift early.

## Proposed Scope
- Add deterministic runtime drift indicators for planner-vs-answer mismatch rates.
- Add rolling window diagnostics and alert thresholds in admin runtime (read-only).
- Expand evaluation with adversarial near-neighbor prompts to stress ambiguous intent routing.
- Keep architecture unchanged (no new LLM agent, no governance-authority mutation).

## Out of Scope
- Broad rollout.
- Normal-user activation policy change.
- KB authority field mutation / Chroma reindex.
