## 5. Drift Guard Module Summary
- Added `PLANNER_DRIFT_GUARD_VERSION = planner_drift_guard_v1`.
- Added deterministic checks for:
  - safety-grounding mechanism drift;
  - short-support length/contact drift;
  - question/practice/revoicing policy drift;
  - close loop reopening;
  - one-step multi-step drift;
  - known-concept deflection drift.
- Added severity mapping (`none|low|medium|high`) and status mapping (`ok|warning|critical`).

## 6. Rolling Monitor Summary
- Added in-memory rolling monitor (`window_size=100`) with:
  - `record_planner_drift_check(user_id, check)`
  - `get_planner_drift_summary()`
  - `reset_planner_drift_summary()`
- Summary stores counters/rates/flag histogram only (no raw user text/full answers).
- Threshold logic implemented:
  - warning: `violation_rate >= 0.10` over `>=20` samples or medium flags in last 5;
  - critical: `critical_rate >= 0.03` over `>=20` samples or high flags in last 3.
