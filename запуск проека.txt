=====================================
5) БЫСТРАЯ ДИАГНОСТИКА ПОСЛЕ ПЕРЕЕЗДА
=====================================
Проверка версий Python:
py -0p

Должно быть:
- 3.12 для bot_psychologist
- 3.10 для Bot_data_base

Если .venv снова "битый" (ошибка про старый путь python.exe):
- удалить .venv
- пересоздать по шагу 0

Если Web UI стартует не на 3000:
- проверь в терминале строку `Local: http://localhost:<порт>/`
- либо освободи порт 3000 и перезапусти `npm run dev`

Если при запуске бота ошибка [Errno 10048] (порт 8001 уже занят):
- проверь занятый порт: `netstat -ano | findstr :8001`
- заверши старый процесс: `taskkill /PID <PID> /F`
- запусти бот заново по шагу 2
